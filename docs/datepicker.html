API

Below are the different public properties and methods accessible to you, the end developer. All of these are accessible on your Datepicker [dp] instance as well as inside event handlers bound to your instance. Some methods have intentionally been left out due to their difficulty tof use for end developers vs. internal use.

Methods

  • instance.setValue(format, value, date)

    Takes a format and value and applies it to the date in question. Returns the formatted value.

    Parameters
    • Name
      format
      Type
      String
      Description
      Piece of a format value (ie: mm, mmmm, yyyy, etc.)
    • Name
      value
      Type
      Number
      Description
      Value to set on a date object.
    • Name
      date
      Type
      Date
      Description
      Date object to pull the value from. Default is internal date.

  • instance.getValue(key[, date])

    Takes a key signifying day, date, month, or year and pulls the value from the date object.

    Parameters
    • Name
      key
      Type
      String
      Description
      Key representing part of a date (ie: m, d, or y).
    • Name
      date
      Type
      Date
      Description
      Date object to pull the value from. Default is internal date.

  • instance.formatValue(format, value)

    Takes a piece of a date format (ie: mm, yyyy, etc.) and formats the raw value.

    Parameters
    • Name
      format
      Type
      String
      Description
      Part of a date format.
    • Name
      value
      Type
      Number
      Description
      Number representing the value to be formatted.

  • instance.parseDate(sdate)

    Takes a date string in *native format only* (yyyy-mm-dd) and converts it to a date.

    Parameters
    • Name
      sdate
      Type
      String
      Description
      Date string in native format (yyyy-mm-dd).

  • instance.dateToString(date[, format])

    Date object to convert to string.

    Parameters
    • Name
      format
      Type
      String
      Description
      format string to convert date to. Default is browser native (yyyy-mm-dd).

  • instance.daysInMonth(date)

    Gets the max days in a perticular date's month.

    Parameters
    • Name
      date
      Type
      Date
      Description
      Date object.

  • instance.daysInLastMonth(date)

    Gets the max days in the month before the date provided.

    Parameters
    • Name
      date
      Type
      Date
      Description
      Date object.

  • instance.daysInNextMonth(date)

    Gets the max days in the month after the date provided.

    Parameters
    • Name
      date
      Type
      Date
      Description
      Date object.

  • instance.startDayInMonth(date)

    Get the start day in the week (1-7).

    Parameters
    • Name
      date
      Type
      Date
      Description
      Date object.

Properties

  • instance.formatmap

    Keeps friendly names for months and days of the week.

  • instance.rootinput

    The input element you provided inside the root datepicker node.

  • instance.input

    Our shadow input created by datepicker. This is the unput the user interacts with.

  • instance.accessbtn

    The button used to access (open/close) the datepicker UI.
