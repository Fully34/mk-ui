Datepicker as Popups

This first example are a set of datepickers in 'standard mode,' which basically means they have a popup or 'dropdown' calendar. Also notice how the data-min and data-max attributes are css selectors. Supplying selectors instead of dates link these two elements together for dynamic min and max dates.
